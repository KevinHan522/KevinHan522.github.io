Project Description

Gravity is a platformer with a gravity-switch mechanic using the up and down keys. You control a small blob that constantly moves forward. Pressing up will set gravity upward and pressing down sets it downward. Hitting any block or obstacle that stops forward movement is an instant game over. Each new screen is completely random and based on different colors and obstacles. Your score goes up over time as well, and the goal is to last as long as possible.

This project was done Fall 2014 in C and compiled as a .gba file. It was a challenge to implement sprite animations smoothly, and I improved on my previous game by using DMA to stop issues with clipping and poor refresh rates. More information on the game can be found in the source link below.

Features

  • Sprite animations
  • DMA rendering
  • Collision detection
