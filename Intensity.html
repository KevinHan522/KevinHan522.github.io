Project Description

Intensity is a semi-bullet hell game where you control a small circle, trying to stay alive as long as possible. Each hit from a bullet will reduce your life count by 1. Enemies will spawn over time and shoot bullets at you, each with their own special characteristic, ranging from shotgun attacks to bouncing shots. Running into them will eliminate them, but since there is always a risk they will shoot you, you can press A to use brief invincibility. B will also speed up the player. Over time, if enemies are not eliminated, they will shoot and move faster and some upgrade their shot type. The higher the score, the faster they spawn as well.

This project was done in Fall 2014. The entire game is coded in base C and can be compiled into a .gba file which can be played on an emulator. More details can be found in the source code

Features

  • Increased AI difficulty over time
  • Collision detection
  • Enemy and bullet variance
